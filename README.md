# C@KE - 온라인 생일 축하 웹서비스
</div>

## 📝 소개
- 현실 친구부터 인터넷 너머의 사람들까지, 특별한 생일날을 보내고 싶은 모든 사람들을 위한 '생일 축하 롤링 페이퍼' 웹 서비스입니다.
- 2024 창업 동아리에서 시제품으로 제작한 프로젝트입니다.
- 📆 개발 기간 : 2024.07 ~ 2024.8
